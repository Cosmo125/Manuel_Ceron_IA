# Módulo 6: Proyectos Prácticos Integradores

## 🎯 Objetivos del Módulo

- Integrar conocimientos de todos los módulos anteriores
- Desarrollar proyectos completos de principio a fin
- Aprender sobre despliegue de modelos en producción
- Crear un portafolio profesional de proyectos de IA

## 📚 Contenido

### Teoría
- `01_metodologia_proyectos_ia.ipynb` - CRISP-DM y metodologías ágiles
- `02_gestion_datos_produccion.ipynb` - Pipelines de datos y MLOps
- `03_despliegue_modelos.ipynb` - APIs, Docker, Cloud deployment
- `04_monitoreo_modelos.ipynb` - Monitoring y mantenimiento de modelos
- `05_etica_responsabilidad.ipynb` - IA responsable y consideraciones éticas

### Ejercicios
- `ejercicio_1_pipeline_completo.ipynb` - Pipeline de ML end-to-end
- `ejercicio_2_api_modelo.ipynb` - Creación de API para modelo
- `ejercicio_3_dashboard.ipynb` - Dashboard interactivo con Streamlit

### Proyectos
- `proyecto_sistema_recomendacion.ipynb` - Sistema de recomendaciones completo
- `proyecto_asistente_virtual.ipynb` - Asistente virtual multimodal
- `proyecto_analisis_redes_sociales.ipynb` - Análisis de sentimientos en redes
- `proyecto_prediccion_financiera.ipynb` - Predicción de mercados financieros
- `proyecto_final_libre.ipynb` - Proyecto personalizado del estudiante

### Datasets
- `ecommerce_data/` - Datos de comercio electrónico
- `social_media_data/` - Datos de redes sociales
- `financial_data/` - Datos financieros históricos
- `custom_datasets/` - Datasets para proyectos personalizados

## 📋 Estructura de Proyectos

Cada proyecto incluye:
- **Definición del problema** y objetivos
- **Análisis exploratorio** de datos
- **Desarrollo del modelo** paso a paso
- **Evaluación y validación** rigurosa
- **Despliegue y producción** (cuando aplique)
- **Documentación completa** del proceso

## ⏱️ Duración Estimada
6-8 semanas

## 📋 Prerrequisitos
- Todos los módulos anteriores completados
- Conocimientos de desarrollo web básico (opcional)

## 🎓 Evaluación
- Proyectos individuales: 70%
- Proyecto final: 30%

## 🚀 Entregables Finales
- Portafolio de proyectos en GitHub
- Presentación de proyecto final
- Documentación técnica completa
- Demo funcional (cuando aplique)
