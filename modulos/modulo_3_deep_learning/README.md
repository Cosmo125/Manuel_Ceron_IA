# Módulo 3: Deep Learning y Redes Neuronales

## 🎯 Objetivos del Módulo

- Comprender los fundamentos de las redes neuronales
- Aprender sobre diferentes arquitecturas de Deep Learning
- Implementar redes neuronales con TensorFlow/PyTorch
- Aplicar técnicas de optimización y regularización

## 📚 Contenido

### Teoría
- `01_introduccion_redes_neuronales.ipynb` - Perceptrón y redes multicapa
- `02_backpropagation.ipynb` - Algoritmo de retropropagación
- `03_funciones_activacion.ipynb` - ReLU, Sigmoid, Tanh, etc.
- `04_tensorflow_keras.ipynb` - Introducción a TensorFlow y Keras
- `05_redes_convolucionales.ipynb` - CNN para imágenes
- `06_redes_recurrentes.ipynb` - RNN, LSTM, GRU
- `07_regularizacion.ipynb` - Dropout, Batch Normalization
- `08_optimizadores.ipynb` - Adam, SGD, RMSprop

### Ejercicios
- `ejercicio_1_perceptron.ipynb` - Implementación desde cero
- `ejercicio_2_mlp_keras.ipynb` - Red multicapa con Keras
- `ejercicio_3_cnn_cifar10.ipynb` - CNN para clasificación de imágenes
- `ejercicio_4_rnn_texto.ipynb` - RNN para análisis de sentimientos

### Proyectos
- `proyecto_reconocimiento_digitos.ipynb` - Clasificador de dígitos MNIST
- `proyecto_generador_texto.ipynb` - Generador de texto con LSTM

### Datasets
- `mnist/` - Dataset de dígitos manuscritos
- `cifar10/` - Dataset de imágenes CIFAR-10
- `sentiment_data.csv` - Datos de análisis de sentimientos

## ⏱️ Duración Estimada
5-6 semanas
