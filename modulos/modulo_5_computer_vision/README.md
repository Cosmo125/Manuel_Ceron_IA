# Módulo 5: Computer Vision

## 🎯 Objetivos del Módulo

- Comprender los fundamentos de la visión por computadora
- Aprender técnicas de procesamiento de imágenes
- Implementar modelos de clasificación y detección de objetos
- Explorar aplicaciones avanzadas como segmentación y generación de imágenes

## 📚 Contenido

### Teoría
- `01_introduccion_computer_vision.ipynb` - Fundamentos de visión artificial
- `02_procesamiento_imagenes.ipynb` - OpenCV y manipulación de imágenes
- `03_extraccion_caracteristicas.ipynb` - SIFT, SURF, HOG
- `04_cnn_vision.ipynb` - Redes convolucionales para visión
- `05_transfer_learning.ipynb` - Uso de modelos preentrenados
- `06_deteccion_objetos.ipynb` - YOLO, R-CNN
- `07_segmentacion.ipynb` - Segmentación semántica y de instancias
- `08_gans_imagenes.ipynb` - Generación de imágenes con GANs

### Ejercicios
- `ejercicio_1_opencv_basico.ipynb` - Operaciones básicas con OpenCV
- `ejercicio_2_filtros_imagenes.ipynb` - Aplicación de filtros
- `ejercicio_3_clasificacion_imagenes.ipynb` - Clasificador de imágenes
- `ejercicio_4_deteccion_rostros.ipynb` - Detector de rostros

### Proyectos
- `proyecto_reconocimiento_objetos.ipynb` - Sistema de reconocimiento
- `proyecto_analisis_medico.ipynb` - Análisis de imágenes médicas
- `proyecto_arte_neural.ipynb` - Transferencia de estilo artístico

### Datasets
- `objects_dataset/` - Imágenes de objetos categorizados
- `medical_images/` - Imágenes médicas de ejemplo
- `faces_dataset/` - Dataset de rostros

## ⏱️ Duración Estimada
5-6 semanas

## 📋 Prerrequisitos
- Módulo 3 completado
- Conocimientos básicos de geometría
