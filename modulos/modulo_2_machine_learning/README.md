# Módulo 2: Machine Learning Fundamentals

## 🎯 Objetivos del Módulo

- Entender los fundamentos del Machine Learning
- Aprender sobre algoritmos supervisados y no supervisados
- Dominar el preprocesamiento de datos
- Implementar modelos básicos de ML

## 📚 Contenido

### Teoría
- `01_introduccion_ml.ipynb` - ¿Qué es Machine Learning?
- `02_tipos_aprendizaje.ipynb` - Supervisado, no supervisado y por refuerzo
- `03_preprocesamiento_datos.ipynb` - Limpieza y preparación de datos
- `04_algoritmos_clasificacion.ipynb` - KNN, Decision Trees, SVM
- `05_algoritmos_regresion.ipynb` - Linear Regression, Polynomial Regression
- `06_clustering.ipynb` - K-Means, Hierarchical Clustering
- `07_evaluacion_modelos.ipynb` - Métricas y validación cruzada

### Ejercicios
- `ejercicio_1_sklearn_basico.ipynb` - Primeros pasos con scikit-learn
- `ejercicio_2_clasificacion.ipynb` - Modelo de clasificación completo
- `ejercicio_3_regresion.ipynb` - Modelo de regresión completo
- `ejercicio_4_clustering.ipynb` - Análisis de clusters

### Proyectos
- `proyecto_prediccion_precios.ipynb` - Predicción de precios de casas
- `proyecto_clasificacion_emails.ipynb` - Clasificador de spam

### Datasets
- `housing_data.csv` - Datos de precios de viviendas
- `email_spam.csv` - Dataset de emails spam/no spam

## ⏱️ Duración Estimada
4-5 semanas

## 📋 Prerrequisitos
- Módulo 1 completado
- Python básico
- Conceptos básicos de estadística
