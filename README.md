# Manuel Cerón - Curso de Inteligencia Artificial

¡Bienvenido al curso de Inteligencia Artificial de Manuel Cerón! Este repositorio contiene todos los materiales, ejercicios y proyectos del curso.

## 📚 Estructura del Curso

### Módulos
- **Módulo 1**: Introducción a la Inteligencia Artificial
- **Módulo 2**: Machine Learning Fundamentals
- **Módulo 3**: Deep Learning y Redes Neuronales
- **Módulo 4**: Procesamiento de Lenguaje Natural (NLP)
- **Módulo 5**: Computer Vision
- **Módulo 6**: Proyectos Prácticos

## 🚀 Requisitos

- Python 3.8 o superior
- pip (gestor de paquetes de Python)
- Git

## 📦 Instalación

1. Clona este repositorio:
```bash
git clone https://github.com/Cosmo125/Manuel_Ceron_IA.git
cd Manuel_Ceron_IA
```

2. Instala las dependencias:
```bash
pip install -r requirements.txt
```

## 📖 Cómo usar este repositorio

Cada módulo tiene su propia carpeta con:
- 📝 **Teoría**: Notebooks con explicaciones conceptuales
- 💻 **Ejercicios**: Actividades prácticas para reforzar el aprendizaje
- 🔬 **Proyectos**: Aplicaciones prácticas de los conceptos aprendidos
- 📊 **Datasets**: Datos para los ejercicios y proyectos
